Installation
============

Requirements
------------

* Python 3.11 or higher
* Poetry (for development)

Create Sandbox
--------------

It's recommended that you install and run the Hydra Router software in a virtual
environment.

.. code-block:: bash

    # Create and activate virtual environment
    python3 -m venv hydra_venv
    source hydra_venv/bin/activate  # On Windows: hydra_venv\Scripts\activate

Installing from PyPI
--------------------

Simply install the **Hydra Router** software using *pip* from within your newly created
virtual environment.

.. code-block:: bash

    . hydra_venv/bin/activate
    pip install hydra-router
